全ての Vue アプリケーション は、Vue 関数で新しい Vue インスタンスを作成することによって起動されるよ。

var vm = new Vue({
  // オプション
});

コンストラクタには引数にオプションオブジェクトを渡せるよ。 オプションオブジェクトは、UI の状態(データ)、状態と DOM マッピングの定義(テンプレート)、マウントさせる DOM 要素、イベントが発火したときに呼び出す振る舞い(メソッド)などを指定します。
